# Ecos-del-portal
Versión Demo jugable del videojuego.

// Repositorio de prueba, es indispensable tener cuenta creada y una pestaña abierta de github. para colaborar con el codigo hacer lo siguiente:
1- instalar github de escritorio de https://desktop.github.com/download/
2- luego de instalar, entrar en la versión de escritorio y File > Clone Repository y buscar ecos del portal
3- En Unity Hub, darle a Add y seleccionar la carpeta que se acaba de descargar
4- ya se puede trabajar, con la versión del escritorio abierta para guardar cada cambio.
5- utilizar ramas si es necesario.

Utilizar gemini para mayor explicación. esta es una guía rapida.
